Qu'est-ce que c'est?

d-portal.org vous aide à explorer les données IATI en fournissant des informations par pays ou par l'éditeur sur les activités de développement et les budgets publiés à IATI.

Il vise à fournir des ministères de tutelle , les parlementaires et la société civile des informations qui peuvent aider à la planification et au suivi des activités de développement.

Dans sa première itération , il contient des données actuelles publiées par l'Initiative internationale pour la transparence de l'aide ( IATI ), ainsi que les plus récentes (2012) données publiées par la créanciers du CAD de l'OCDE système de notification.

La plate-forme actuelle pose deux questions : d'où viennent les ressources proviennent , et où vont-ils ( par secteur et localisation) ? Des données récentes , en cours et à venir est présenté et comparé différentes sources de données.

1. D'où proviennent les ressources? 2. Où vont-ils (par secteur et localisation)?

Des données récentes, actuelles et futures est présenté et comparé les sources de données.

d-portal.org est une plate-forme open source. Cela signifie que vous pouvez utiliser tous les aspects de la plate-forme pour construire votre propre portail en utilisant le code source libre.

Le concept open source repose sur le partage des ressources dans l'espoir que d'autres se joindront et construire une meilleure plate-forme ensemble. Cela comprend la recherche et signaler les bogues, contribuant de nouvelles fonctionnalités que vous souhaitez la plate-forme d'avoir ou même de créer de nouveaux thèmes afin que les autres puissent l'utiliser.

Generator widgets pour votre site web

Il y a une énorme quantité d'informations intéressantes au sein de l'IATI et la valeur significative de pouvoir partager bouchées de librement sur d'autres sites et blogs. Le générateur d-portail permet aux utilisateurs de sélectionner des informations spécifiques à partir d-portail et intégrer directement dans leur propre code de site.

Une fois que vous avez intégré tout contenu qu'il mettra automatiquement à jour (la nuit) que les données des éditeurs est mis à jour dans le registre de IATI. Lorsque les utilisateurs cliquent sur le contenu, ils seront dirigés vers la page correspondante sur le d-portail afin qu'ils puissent étudier plus.

Generator est actuellement accessible à http://www.d-portal.org/ctrack.html?#view=generator

Savi est une visionneuse d'activité simple pour l'IATI XML que vous pouvez voir en temps réel, en ligne, dans votre navigateur préféré.

Savi utilise CSS et javascript pour afficher XML autrement complexe regardant dans un format qui est plus facile à lire et à digérer.

dstore subventionne l' IATI Registry avec une version nodejs + PostgreSQL + SQLite optimisé pour une utilisation dans les requêtes en temps réel.
Q vous permet d'interroger les informations dont vous avez besoin via des filtres simples mais complexes.
Voici quelques exemples de requêtes :
Append requêtes avec votre format préféré .json, .xml, .csv or .html
http://d-portal.org/q.xml?limit=10&orderby=day_start&day_start_gt=1950-01-01
Retourne XML pur
http://d-portal.org/q.json?limit=10&orderby=day_start&day_start_gt=1950-01-01
Renvoie JSON
http://d-portal.org/q.html?limit=10&orderby=day_start&day_start_gt=1950-01-01
Retourne XML qui est automatiquement formaté en HTML (SAVi). Source reste pur XML
Cette requête affiche activités de limiter le montant à 10 limit=10 triés en commençant par jour orderby=day_start et à partir de 1950 day_start_gt=1950-01-01.
http://d-portal.org/q.xml?from=act,sector,country&country_code=bd&sector_percent_not_null
Retourne XML pur
Cette requête affiche toutes les activités de Bangladesh (code du pays est BD) country=bd qui ont secteurs signalés dans les données sector_percent_not_null. En fin de compte, cette requête vous permet de regarder toutes les activités qui ont des graphiques en eux.
http://d-portal.org/q?from=act,country&groupby=reporting_ref&select=stats&country_code=BD
Retourne JSON (JSON est par défaut) des données sur les activites (activités sont par défaut) où 100 lignes de la base de données est la limite par défaut
Cette requête affiche des statistiques de données select=stats sur activites pour chaque organisation de rapports groupby=reporting_ref pour le Bangladesh country_code=BD.
http://d-portal.org/q.csv?from=trans&orderby=trans_day&limit=1000
Retours fichier CSV (nommé q.csv par défaut)
Cette requête vous permet de télécharger le CSV q.csv 1000 limit=1000 données de transaction from=trans triés par date orderby=trans_day.

Questions fréquemment posées

d-portail est un site expansive qui contient des portails basés non seulement du pays et de l'éditeur, mais aussi des outils comme Savi, Générateur, Dash et bientôt, Recherche - une interface de recherche simple soutenue par dStore et Q. Avec cela vient d'une base de données de plus en plus de questions sur l'architecture et le fonctionnement des différentes parties du site.

Pour l'instant, nous brisons ces bas à 3 différents types de questions - général, technique et des questions sur le générateur. À l'avenir, nous allons ajouter plusieurs onglets à la liste pour compenser les fonctionnalités à venir. Nous supportons actuellement les traductions en anglais et en français. Si vous souhaitez fournir des traductions pour d'autres langues, nous vous encourageons à nous contacter.

FAQ peut être trouvé à l'adresse http://www.d-portal.org/faq.html

Dash : Editeurs statistiques pour d-portal.org

Dash est un aperçu des données qui peuplent d-portal.org avec un accent principalement sur ​​les statistiques de l'éditeur et la qualité des données déclarées par eux.

En vous montrant où et quand des données non valides est rapporté, Dash sera en mesure d'expliquer les écarts qui se reflètent sur ​​le site principal.

Dash n'est pas pays spécifique de sorte que vous pourrez voir toutes les statistiques des éditeurs disponibles dans tous les pays inclus dans le portail. C'est à accueillir le fait qu'il existe plusieurs éditeurs qui signalent des activités dans plusieurs pays bénéficiaires.

Dash est actuellement accessible à l'adresse http://d-portal.org/ctrack.html?#view=dash. Dash est une nouvelle fonctionnalité et est encore en cours d'élaboration.

Explore

Explore est une nouvelle fonctionnalité qui est actuellement en cours de développement. Soutenu par dStore et Q, il fournira une interface de recherche simple pour les utilisateurs de filtrer à travers la vaste base de données de IATI.

Nous recherchons actuellement des commentaires sur le intuitivité et la convivialité de Explore. N'avoir un aller et envoyez-nous vos commentaires en ajoutant une nouvelle émission à travers le Github forum ou, si vous préférez, entrer en contact par le biais support@iatistandard.org. Nous aimerions entendre ce que vous pensez!

Essayez Explore à http://www.d-portal.org/ctrack.html#view=search

Dquery

Interface SQL en lecture seule vers la base de données complète, dquery fournit aux utilisateurs techniques un accès à toutes les données IATI qui peuplent d-portal et des options pour télécharger les données aux formats XML, JSON, HTML et CSV.

Cette nouvelle fonctionnalité est toujours en cours de test et de mise à jour. Veuillez nous contacter si vous avez des questions ou des suggestions.

Utilisez l'interface à http://www.d-portal.org/dquery

Comment installer d-portal

Au cœur du projet d-portal est le but que n'importe qui n'importe où peut prendre le code open source derrière d-portail, déployer pour eux-mêmes et de créer un portail d-mesure pour un pays qui les intéresse. Cela pourrait inclure l'ajout d'autres données de l'aide des systèmes d'information de gestion, les budgets nationaux et une série d'indicateurs socio-économiques comparables.

d-portal est constitué de la backend dstore et l'interface ctrack. Pour installer d-portal, vous devez configurer un serveur. Commencez à lire des instructions pour installer un serveur d-portal
